## 프로젝트 소개

SwiftUI와 Firebase 공부를 위해 진행한 프로젝트입니다.</br>

초기 공부단계에서 보편적으로 많이 만드는 To Do List App 입니다.</br>
파이어베이스를 이용해 로컬에서만이 아닌 다른 환경에서도 유저데이터를 관리할 수 있게끔 개발했습니다.</br>
